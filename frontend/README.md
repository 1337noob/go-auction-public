# Фронтенд аукциона

Простой фронтенд для системы аукционов на React.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Фронтенд будет доступен по адресу http://localhost:3000

## Использование

1. Введите ID аукциона и ID пользователя на стартовой странице
2. Нажмите "Подключиться"
3. Страница аукциона отобразит:
   - Информацию об аукционе
   - Текущую ставку (если есть)
   - Таймер обратного отсчета таймаута
   - Форму для размещения ставки

## Функционал

- **Получение информации об аукционе** - загрузка данных через API
- **WebSocket подключение** - получение событий в реальном времени
- **Обработка событий:**
  - `AuctionStarted` - разблокирует возможность делать ставки
  - `BidPlaced` - обновляет текущую ставку и таймаут
  - `AuctionCompleted` - завершает аукцион и показывает финальную сумму
- **Блокировка ставок** - до старта аукциона ставки заблокированы
- **Таймер таймаута** - показывает оставшееся время после последней ставки

## API

Фронтенд использует следующие RESTful эндпоинты:

- `GET /api/auctions/{id}` - получение информации об аукционе
- `POST /api/auctions/{id}/bids` - размещение ставки
- `WS /ws?auction_id=...&user_id=...` - WebSocket подключение для событий

## Настройка

В `vite.config.js` настроен прокси для API запросов к бэкенду на порту 8081.
